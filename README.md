# derekwelty.com

Personal website of Derek Welty.

## Stack
- HTML / CSS / JS
- Bootstrap + jQuery (CDN)
- Less → CSS via Gulp
- Hosted on GitHub Pages

## Development
```
npm install
npx gulp
```

## Deploy
Push to `main`. GitHub Pages auto-deploys.
